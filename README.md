После этого я решил получить более фундаментальные знания и успешно закончил c получением сертификата курс cs50 от университета Гарвард. После этого я решил продолжить изучение Frontend, в частности получить углубенные знания Javascript, React, Typescript и другие необходимые технологии. Приступил к изучению курса Frontend от Karpov courses. Параллельно работал на фрилансе и выполнил несколько заказов на верстку лэндингов.
